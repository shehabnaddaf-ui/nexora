# Nexora — Landing Page

Premium landing page for **Nexora Software Solutions**.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Arabic / English (RTL / LTR)

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy

Compatible with Vercel, Netlify, and Cloudflare Pages.

- Build command: `npm run build`
- Output directory: `dist`
